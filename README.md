# TicketFlow Pro

TicketFlow Pro es una plataforma de gestión de tickets pensada para empresas que quieren ordenar su soporte, mejorar tiempos de respuesta y dar visibilidad al cliente en todo momento.

## ¿Qué es?

Una solución SaaS multiempresa donde cada organización puede:

- recibir incidencias de sus clientes
- asignar y gestionar tickets internamente
- seguir el estado de cada caso hasta su cierre
- centralizar la comunicación en un único lugar

## ¿Para quién está pensado?

- equipos de soporte técnico
- departamentos de atención al cliente
- agencias o empresas con varios clientes y alto volumen de solicitudes
- negocios que necesitan trazabilidad y control del servicio

## Problema que resuelve

Cuando las incidencias llegan por correo, chat o mensajes sueltos, se pierde contexto y control. TicketFlow Pro ayuda a:

- evitar tickets duplicados o sin seguimiento
- definir responsables claros
- mantener historial de cada solicitud
- dar al cliente una experiencia más profesional

## Experiencia de uso

La plataforma está organizada en tres perfiles:

- `SuperAdmin`: configura la aplicación y administra cuentas internas
- `Admin`: gestiona el trabajo diario de tickets y clientes
- `Cliente`: accede a su portal para crear incidencias y consultar el progreso

Además, incluye asistente de configuración inicial para dejar el entorno listo desde el primer uso.

## Estado del proyecto

Este repositorio contiene una base sólida y funcional para continuar evolución de producto.

Incluye:

- backend de gestión y autenticación
